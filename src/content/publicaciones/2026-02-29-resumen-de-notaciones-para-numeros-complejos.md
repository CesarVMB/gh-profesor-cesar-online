Las fórmulas correspondientes a las formas de escribir números complejos son equivalentes entre sí y el uso de una u otra dependerá de las condiciones del ejercicio, problema o situación matemática en que nos encontremos.

| Notación                  | Fórmula                       |
| ------------------------- | ----------------------------- |
| Cartesiana (par ordenado) | $z=(a,b)$                     |
| Binomial (canónica)       | $z=a+bi$                      |
| Trigonométrica (polar)    | $z=r(\cos\theta+i\sin\theta)$ |
| Exponencial               | $z=re^{i\theta}$              |
